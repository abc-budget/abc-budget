# MCC reference — attribution

The data in `mcc-reference.json` is vendored (a derived, filtered subset) from:

- **Repository:** [Oleksios/Merchant-Category-Codes](https://github.com/Oleksios/Merchant-Category-Codes)
- **Pinned commit:** `322d99f576bc42c0248b6e83e8902c6cea5d23a6`
- **Upstream license:** MIT — Copyright (c) 2021 Oleksii Teslenko
- **Generated by:** `tools/gen-mcc-reference.mjs` (MANUAL — never run at install/build time)

We keep the `uk` and `en` titles only (the `ru` set is dropped), zero-pad each
code to a 4-digit string, and flatten to `{ "<dddd>": { "uk", "en" } }`. The JSON
is committed and imported directly by `mcc-lookup.ts` — there is ZERO runtime
network and no postinstall/build fetch.
