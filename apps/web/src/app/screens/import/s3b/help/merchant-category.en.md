The **MCC** code (**Merchant Category Code**) is a four-digit code used by payment systems (e.g., Mastercard, Visa, and
American Express) to classify businesses. Each MCC code is assigned to a specific category of business. It helps to
understand what a merchant does (for example, whether it is a grocery store, restaurant, hotel, etc.). In other words,
the MCC code indicates the direction of the business.
This type of data does not require any additional settings.
