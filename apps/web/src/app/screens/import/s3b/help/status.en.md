The **Transaction status** on the bank statement indicates the result of the transaction, for example, "Completed" or "Rejected".
**ABC Budget** analyzes only successfully completed financial transactions, so it is important to identify the status value that indicates this.
### Configuration
#### Successful status of a transaction
In most cases, it is enough to select "**Automatic detection of successful status**". The **ABC Budget** will search for
the most likely values of the transaction status and try to apply them to the transactions. If you see any errors in
determining the status of a transaction, you can specify the transaction status value manually.
