The **Currency** field in a bank statement indicates the currency in which the financial transactions were carried out.
This may be the national currency (e.g., hryvnia, dollar, euro) or another currency. The currency is indicated using a
currency code (e.g., **UAH** for hryvnia, **USD** for US dollar, **EUR** for euro, etc.). Less commonly, the currency
is indicated by a currency symbol (e.g., **$** for US dollar, **€** for euro, etc.).
This type of data does not require any additional settings.
