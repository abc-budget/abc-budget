**Валюта** у банківській виписці вказує на валюту, в якій здійснювалися фінансові операції. Це може бути національна
валюта (наприклад, гривня, долар, євро) або інша валюта. Валюта вказується за допомогою валютного коду (наприклад,
**UAH** для гривні, **USD** для долара США, **EUR** для євро тощо). Рідше валюта вказується за допомогою символу
валюти (наприклад, **$** для долара США, **€** для євро тощо).
Цей тип даних не потребує жодних додаткових налаштувань.
