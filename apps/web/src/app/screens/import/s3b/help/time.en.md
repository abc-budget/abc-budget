**Time** indicates the time of day a financial transaction was made (e.g. 10:30). Some banks provide it in
a separate column (apart from the date).
ℹ️ **Important**: when processing operations, **ABC Budget** relies only on the **date** — the **time is
not used** and is not stored. This is a deliberate privacy choice (so as not to pin your operations to a
precise moment).
You can mark a time column as «**Time**» to label it explicitly: its values are recognized but **won't be
used**. You may also simply «**Ignore**» the column — the result is the same.
This data type needs no additional configuration.
