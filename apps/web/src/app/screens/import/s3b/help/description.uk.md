**Опис** операції у банківській виписці — це текстовий опис або коментар, який пояснює природу або деталі фінансової операції. В цій колонці можуть бути вказані додаткові відомості про операцію, такі як назва магазину, отримувач платежу, номер рахунку тощо.
Цей тип даних не потребує жодних додаткових налаштувань.
