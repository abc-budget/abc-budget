Banks may charge a service fee in general or for certain individual transactions. Some banks display this data as a
separate transaction on the account. In this case, they are analyzed in the same way as any other transaction,
according to your "_rules_".
If the transaction fee is displayed as an additional column, i.e. it is part of a single transaction, you can select
the column type "**Bank Commission**". In this case, **ABC Budget** will analyze it separately from the main transaction.
### Configuration
#### Currency
> **Base currency** is the currency you use in your everyday life and most financial transactions are made in this
> currency. You can change the base currency in the settings.
* **Auto-detect** - If the statement has a separate currency column, it will be used. Otherwise, the base currency will be used.
* **Use Base Currency** - The base currency will be used.
* **Select Currency** - You can select a currency from the list.
