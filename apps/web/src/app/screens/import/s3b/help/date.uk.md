**Дата** у банківській виписці вказує на день, коли була здійснена фінансова операція. Це може бути дата покупки,
переказу коштів, зняття готівки з банкомату тощо.
**Дата** у банківській виписці зазвичай відображається у **стандартному форматі**, який включає день, місяць та рік.
Зазвичай це виглядає як **dd.MM.yyyy** (наприклад, 15.11.2023). Деякі банки також можуть вказувати час операції,
який додається до дати у форматі **dd.MM.yyyy HH:mm** (наприклад, 15.11.2023 10:30). Важливо враховувати, що формат
дати може відрізнятися в залежності від банку та регіону.
ℹ️ **Важливе зауваження**: **ABC Budget** при обробці операцій спирається лише на дату, час операції не враховується.
Його можно не вказувати у налаштуваннях.
### Конфігурація
#### Формат дати
У більшості випадків достатньо вибрати "**Автоматичне визначення формата дати**". **ABC Budget** перебере найбільш
імовірні формати дати та спробує застосувати їх до операцій. Якщо ви побачили помилки у визначенні дати, ви можете
вказати формат дати вручну.
⚠️**Важливо**: Формат дати вказується у [форматі "Unicode Technical Standard #35
"](https://www.unicode.org/reports/tr35/tr35-dates.html#Date_Field_Symbol_Table).
Якщо у даті використовуються фіксовані символи (тобто вони не залежать від дати та часу), наприклад префікс "Дата :",
то їх потрібно вказати у форматі дати взявши в одинарні дужки (') окрім символів пунктуації. Наприклад, якщо дата
виглядає як "Дата : 15.11.2023", то формат дати буде виглядати як "'Дата :' dd.MM.yyyy".
| **Шаблон** | **Опис**   | **Приклад**    |
|------------|------------|----------------|
| y          | Рік        | 2023           |
| yy         |            | 23             |
| yyyy       |            | 2023           |
| M          | Місяць     | 1, 2, ... 12   |
| MM         |            | 01, 02, ... 12 |
| MMM        |            | листоп.        |
| MMMM       |            | листопада      |
| d          | День       | 1, 2, ... 31   |
| dd         |            | 01, 02, ... 31 |
| E          | День тижня | пн, вт, ... нд |
| EEEE       |            | понеділок      |
