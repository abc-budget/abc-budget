**Cashback** is the amount of money that a bank returns to a customer for certain transactions. For example, if a
customer makes a purchase with a bank card, part of the cost can be returned to their account as cashback. This can be
a percentage of the purchase amount or a fixed amount. Cashback can be indicated separately in the statement as a
transaction with a special code or simply as a separate column.
### Configuration
#### Currency
> **Base currency** is the currency you use in your everyday life and most financial transactions are made in this
> currency. You can change the base currency in the settings.
* **Auto-detect** - If the statement has a separate currency column, it will be used. Otherwise, the base currency will be used.
* **Use Base Currency** - The base currency will be used.
* **Select Currency** - You can select a currency from the list.
