> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Vendored currency reference (display/format/reverse-lookup, uk/en), the ENT-011 resolution API, engine-internal USD normalization, the 2-level rate cache (local IDB → injected remote api), and the hardened date-only OER-proxy function — with the engine's runtime public surface unchanged.

The function PORTS prior-art `getUSDRates` (`D:\abc-budget\.deprecated\firebase\functions\src\index.ts`) with 4 adaptations: auth→App Check, +budget cap, +date range, CORS. Migration v2 adds the first real store.

**Tech Stack:** vendored JSONs (no registry), Intl.NumberFormat with explicit fraction digits, firebase SDK (app/functions/app-check) in apps/web only, firebase-functions v2 + vitest in standalone `functions/`, fake-indexeddb for cache tests.
