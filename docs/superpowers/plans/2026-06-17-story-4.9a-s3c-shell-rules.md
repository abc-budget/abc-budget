## The v4 wire contract (pinned ‚Äî all tasks bind to this)
`CONTRACT_VERSION = 4`. Add to `EngineMethod`: `importCategorizedRows`, `importConditionFields`, `importWhy`,
`importRulesList`, `rulesCreate`, `categoriesList`, `categoriesCreate`. DTOs (serializable; in `client/dto.ts`):
```ts
CategoryDTO          = { id: string; name: string; icon: string; currency: string };       // ¬´base¬ª resolved
ConditionDTO         = { field: string; operator: string; value: unknown; currency?: string };
ConditionFieldDTO    = { field: string; valueKind: 'text'|'num'|'num2'|'day'|'day2'|'code'|'optone'|'optset'|'regex'|'bool';
                         operators: string[]; options?: { value: string; label: string }[] };
CategorizedRowDTO    = { rowIndex: number; date: string; amount: number; currency: string; description: string|null;
                         counterparty: string|null; account: string|null; bankCategory: string|null; mcc: number|null;
                         categoryId: string|null; isManual: 0|1; ruleId: number|null;
                         previousCategoryId?: string|null; atypical?: unknown };   // last two: 4.9b/c seams (unused here)
CategorizedWindowDTO = { rows: CategorizedRowDTO[]; total: number; matchCount: number };  // matchCount = total draft matches across ALL rows
WhyRuleDTO           = { ruleId: number; status: 'win'|'miss'|'neutral';
                         conditions: { field: string; operator: string; value: unknown; met: boolean|null }[]; categoryId: string };
WhyTreeDTO           = { manual: { categoryId: string } | null; rules: WhyRuleDTO[]; winnerRuleId: number|null };
RuleSummaryDTO       = { ruleId: number; conditions: ConditionDTO[]; categoryId: string; appliedCount: number };
```
Method signatures:
```ts
importCategorizedRows(sessionId, opts: { offset: number; count: number; segment: 'all'|'uncat'; draft?: ConditionDTO[] }): Promise<CategorizedWindowDTO>;
importConditionFields(sessionId): Promise<ConditionFieldDTO[]>;
importWhy(sessionId, rowIndex): Promise<WhyTreeDTO>;
importRulesList(sessionId): Promise<RuleSummaryDTO[]>;
rulesCreate(conditions: ConditionDTO[], categoryId: string): Promise<{ ruleId: number }>;
categoriesList(): Promise<CategoryDTO[]>;
categoriesCreate(input: { name: string; icon: string; currency: string }): Promise<CategoryDTO>;
```
