# Story 5.4 — S3d Review & Commit UI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the S3d import-review screen (the 4th wizard step) — pixel-faithful to the design bundle but bound to the REAL v8 engine wire (`importReview` / `importCommit`), with fail-loud error ack, dedup display, and commit-then-discard-raw.

**Architecture:** Mirror the S3c integration exactly. A `useS3dSession(client, sessionId, active)` hook loads ONE `importReview` window + `categoriesList` when the step is reached, exposes `{window, summary, rows, categoryIndex, filter, page, ack, phase, hasErrors, canSave, commit, rowsCommitted}`, and drives the review→saving→saved phase machine. A prop-driven `S3dReview` view renders the table + summary tiles + dedup bar (ALL counts from `window.summary`, never recomputed) and the saved overlay. `ImportFlow` wires the hook, renders `S3dReview` for step 4, and makes the footer phase-aware (review: Back + ack + «Зберегти N»; saved: «Імпортувати ще»/«До бюджету»). Pure consumer of v8 — NO contract bump.

**Tech Stack:** React 19 + react-router v7, TypeScript, Vitest + @testing-library/react, the `@abc-budget/engine` barrel (v8 DTOs), the Altus i18n catalog (`useT`/`useLang`).

## Global Constraints

- **Bind to the DTO, NOT the prototype.** Real-DTO fixtures ONLY (`ReviewWindowDTO`/`ReviewSummaryDTO`/`ReviewRowDTO`/`CommitResultDTO`). The prototype's `s3d-data.jsx` shapes (`r.id`/`r.cur`/`r.cat`/`r.sub`/`r.cp`/`r.mcc`) are NOT the contract.
- **Drift trap 1 — summary is wire-authoritative.** ALL tiles, the dedup bar (`newCount`/`dup`), `saveCount`, and `hasErrors` come from `window.summary` (full-set). NEVER recompute counts from the `rows` array (it's a window).
- **Drift trap 2 — reasons are pre-serialized.** Render `row.reasons: SerializedMessage[]` through the shared `resolveMessage` (Task 1). NO `r.sub` subcode switch.
- **Drift trap 3 — category via `categoryId` + `categoriesList()`.** Build an `id→CategoryDTO` index; render name + icon from it; `categoryId == null` → the «—» (catNone) cell.
- **Drift trap 4 — label column = `description`.** No cp/mcc/desc field-picking; the DTO has only `description`, under the `colDesc` header.
- **Drift trap 5 — `currency` not `r.cur`; `rowIndex` (React key) not `r.id`; `date` is a full ISO string** — format the compact display from ISO.
- **Drift trap 6 — saved count = `CommitResultDTO.rowsCommitted`**, not the pre-commit `newCount`.
- **dup ≠ skipped.** A dup is `state:'ok' + dup:true` (NOT re-stated as skipped). Mark it `is-dup`, count it in `summary.dup`, exclude it from `newCount`. (The prototype conflated dup→skipped; we don't.)
- **amount:** `ok` is UNSIGNED; `error`/`skipped` echo RAW (income = skipped, raw signed). `null` amount/currency → «—». No abs, no re-detect in the UI.
- **HC-6** — chrome strings are translated; user content (descriptions, reasons' `{text}`) is NEVER translated.
- **No contract bump** — `CONTRACT_VERSION` STAYS 8 (pure v8 consumer). Confirm in the gate.
- **Full cold gate (the 5.2 lesson):** `web tsc --noEmit` (0) · `tsc -b` (0) · engine vitest · web vitest · functions vitest · eslint (0). A green vitest/build is NOT proof of types.
- **Reuse, don't recreate:** the Altus chrome (FlowHeader/Stepper/LangToggle/Panel/Key/Lamp), `fmtAmount` (`s3c/money.ts`), `CatIcon` (`ui/altus/icons`), `useT`/`useLang`, the router (`useNavigate`).

export type S3dFilter = 'all' | 'error' | 'skip' | 'both';
export type S3dPhase = 'review' | 'saving' | 'saved';
export interface S3dSession {
  readonly window: ReviewWindowDTO;
  readonly summary: ReviewSummaryDTO;       // === window.summary (authoritative)
  readonly rows: readonly ReviewRowDTO[];   // === window.rows (the loaded window)
  readonly categoryIndex: Map<string, CategoryDTO>;
  readonly filter: S3dFilter;
  setFilter(f: S3dFilter): void;
  readonly page: number;
  setPage(p: number): void;
  readonly ack: boolean;
  setAck(a: boolean): void;
  readonly phase: S3dPhase;
  readonly hasErrors: boolean;              // summary.error > 0
  readonly canSave: boolean;                // !hasErrors || ack
  commit(): Promise<void>;                  // importCommit → saving→saved; sets rowsCommitted
  readonly rowsCommitted: number;
}
export function useS3dSession(client: EngineClient, sessionId: string, active?: boolean): S3dSession;

// S3dReview.tsx
export function S3dReview({ session }: { session: S3dSession }): JSX.Element;
```

**Constants:** `WINDOW_COUNT = 5000` (one window covers any realistic statement; the engine returns `min(count, total)`; `summary` stays full-set authoritative regardless). `PG = 14` (page size, from the design).

**Phase machine:** `commit()` guards on `canSave && phase==='review'`; sets `phase='saving'`; on `importCommit` success sets `rowsCommitted` + `phase='saved'`; on failure resets `phase='review'` and rethrows (fail-loud — the session is NOT freed on `RatesUnavailableError`, so retry is possible).

**Re-entry refetch (divergence from S3c):** S3d re-loads the window every time `active` transitions true (deps `[client, sessionId, active]`, no `seededSessionRef` guard) and resets `filter='all'`, `page=0`, `ack=false`, `phase='review'` — because edits in S3c upstream change the review. (S3c keys on sessionId; S3d must reflect upstream edits on re-entry.)

import type { SerializedMessage } from '@abc-budget/engine';
import type { ChromeKey } from './i18n';
import { t as translate } from './i18n';

/**
 * Resolve a SerializedMessage to a display string.
 * Native (`{text}`) → text verbatim (HC-6 — user content is never translated).
 * Localizable (`{key, params}`) → the chrome catalog rendering when the key
 * exists there, else the raw key (best-effort — engine keys are not all in the
 * web chrome catalog).
 */
export function resolveMessage(msg: SerializedMessage, lang: 'uk' | 'en'): string {
  if ('text' in msg) return msg.text;
  try {
    const rendered = translate(lang, msg.key as ChromeKey, msg.params as Record<string, string | number>);
    return rendered ?? msg.key;
  } catch {
    return msg.key;
  }
}
```

import type {
  CategoryDTO, EngineClient, ReviewSummaryDTO, ReviewWindowDTO,
} from '@abc-budget/engine';

export type S3dFilter = 'all' | 'error' | 'skip' | 'both';
export type S3dPhase = 'review' | 'saving' | 'saved';

export interface S3dSession {
  readonly window: ReviewWindowDTO;
  readonly summary: ReviewSummaryDTO;
  readonly rows: readonly ReviewWindowDTO['rows'][number][];
  readonly categoryIndex: Map<string, CategoryDTO>;
  readonly filter: S3dFilter;
  setFilter(f: S3dFilter): void;
  readonly page: number;
  setPage(p: number): void;
  readonly ack: boolean;
  setAck(a: boolean): void;
  readonly phase: S3dPhase;
  readonly hasErrors: boolean;
  readonly canSave: boolean;
  commit(): Promise<void>;
  readonly rowsCommitted: number;
}

const EMPTY_SUMMARY: ReviewSummaryDTO = { total: 0, ok: 0, error: 0, skipped: 0, dup: 0, newCount: 0 };
const EMPTY_WINDOW: ReviewWindowDTO = { summary: EMPTY_SUMMARY, rows: [] };
/** One window covers any realistic statement; the engine returns min(count,total).
 *  summary stays full-set authoritative regardless of the window size. */
const WINDOW_COUNT = 5000;

export function useS3dSession(client: EngineClient, sessionId: string, active = true): S3dSession {
  const [reviewWindow, setReviewWindow] = useState<ReviewWindowDTO>(EMPTY_WINDOW);
  const [categories, setCategories] = useState<CategoryDTO[]>([]);
  const [filter, setFilter] = useState<S3dFilter>('all');
  const [page, setPage] = useState(0);
  const [ack, setAck] = useState(false);
  const [phase, setPhase] = useState<S3dPhase>('review');
  const [rowsCommitted, setRowsCommitted] = useState(0);

  // Re-fetch on EVERY active-true transition (S3c upstream edits change the review).
  useEffect(() => {
    if (!active) return;
    if (!sessionId) { setReviewWindow(EMPTY_WINDOW); setCategories([]); return; }
    let live = true;
    setFilter('all'); setPage(0); setAck(false); setPhase('review');
    void (async () => {
      const [win, cats] = await Promise.all([
        client.importReview(sessionId, { offset: 0, count: WINDOW_COUNT }),
        client.categoriesList(),
      ]);
      if (!live) return;
      setReviewWindow(win);
      setCategories(cats);
    })();
    return () => { live = false; };
  }, [client, sessionId, active]);

  const categoryIndex = useMemo(() => new Map(categories.map((c) => [c.id, c])), [categories]);
  const summary = reviewWindow.summary;
  const hasErrors = summary.error > 0;
  const canSave = !hasErrors || ack;

  const phaseRef = useRef(phase);
  phaseRef.current = phase;
  const commit = useCallback(async () => {
    if ((hasErrors && !ack) || phaseRef.current !== 'review') return;
    setPhase('saving');
    try {
      const res = await client.importCommit(sessionId);
      setRowsCommitted(res.rowsCommitted);
      setPhase('saved');
    } catch (err) {
      setPhase('review'); // fail-loud: session not freed on RatesUnavailableError → retry-able
      throw err;
    }
  }, [client, sessionId, hasErrors, ack]);

  return {
    window: reviewWindow, summary, rows: reviewWindow.rows, categoryIndex,
    filter, setFilter, page, setPage, ack, setAck, phase, hasErrors, canSave, commit, rowsCommitted,
  };
}
