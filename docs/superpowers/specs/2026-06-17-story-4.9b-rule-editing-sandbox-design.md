## The 4.5 engine model (what we wire — already merged)
`RuleSandboxSession` IS the single edit funnel: `classify(action): 'sandbox'|'live'` (sync, O(1)) ·
`submit(action)` (async — **live: apply to current + persist NOW; sandbox/engaged: accumulate into the
virtual tree**) · `computeDiff(): DiffRow[]` (sync, override-ops structurally excluded) · `apply()`
(saveDecisionTree delta → virtual becomes current) · `cancel()` (discard) · `get engaged`. `EditAction =
reorder | delete | editConditions | categoryOnly | appendEnd`. `DiffRow = { rowIndex, oldCategoryId,
newCategoryId }`. Deps = `{ importRows, overrideMap, categoriesById, currentTree, persistence }` — exactly the
ctx `importCategorizedRows` already builds (rows + `reload()` + `loadOverrideMap` + categoriesById).

### The v5 wire methods (signatures)
```ts
rulesClassify(sessionId, action: EditActionDTO): Promise<'live' | 'sandbox'>;   // pure (dynamic-button preview); no retained session
sandboxSubmit(sessionId, action: EditActionDTO): Promise<SandboxStateDTO>;       // funnel: live→persist+discard / sandbox→engage+retain
sandboxDiff(sessionId): Promise<SandboxStateDTO>;                                 // current state (re-fetch / «Переглянути зміни»)
sandboxApply(sessionId): Promise<void>;                                          // saveDecisionTree delta; virtual→current; teardown
sandboxCancel(sessionId): Promise<void>;                                         // discard; teardown
// SandboxStateDTO = { engaged: boolean; count: number; changed: DiffRowDTO[] }
// DiffRowDTO     = { rowIndex; oldCategoryId: string|null; newCategoryId: string|null }
// EditActionDTO  = { kind:'reorder'; order:number[] } | { kind:'delete'; ruleId } | { kind:'categoryOnly'; ruleId; categoryId }
//                | { kind:'editConditions'; ruleId; before: ConditionDTO[]; after: ConditionDTO[] } | { kind:'appendEnd'; conditions: ConditionDTO[]; categoryId }
```
The worker maps `EditActionDTO → EditAction` via `conditionsToRules` (4.1 factories; amount via
`createAmountCondition`) + `categoriesById` lookup (categoryId → Category) + the persisted rule's id.

### `importCategorizedRows` becomes SANDBOX-AWARE (the OPS old→new) [confirm]
